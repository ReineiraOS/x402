# @reineira-os/x402-contracts

Deployable RSS implementations — the delivery-deadline resolver, delivery policy, and the x402 /
confidential x402 escrow receivers. Conformance-tested against
[`@reineira-os/rss`](https://www.npmjs.com/package/@reineira-os/rss).

> Part of [**x402**](https://github.com/ReineiraOS/x402) — insured, FHE-confidential settlement on the [Reineira Settlement Standard](https://www.npmjs.com/package/@reineira-os/rss).

## Install

```bash
pnpm add @reineira-os/x402-contracts
```

Ships Solidity sources (`contracts/`) and deploy scripts (`script/`). Solidity `0.8.25` (cancun).
